# 🍽 Restaurant POS System with QR Code Payment

A Python Tkinter based Restaurant Billing System with QR Code Payment integration.

## 🚀 Features

- Customer Details
- Bill Number Generation
- Menu Management
- QR Code Payment
- Automatic Bill Receipt
- GST Calculation
- Save Bills in TXT File

## 🛠 Technologies Used

- Python
- Tkinter
- Pillow
- qrcode
- File Handling

## ▶️ Run Project

Install modules:

```bash
pip install pillow qrcode
```

Run project:

```bash
python cafe_qr.py
```

## 📷 Project Output

Restaurant POS GUI Billing System with QR payment support.
